## 📊 統一的數據分析框架

### 第 1 步：數據收集

**所有問卷統一的收集方式**：
```
1. 使用 Google Forms
2. 設定截止日期（通常 7-14 天）
3. 每 3 天發送一次提醒
4. 導出 Excel 數據
```

### 第 2 步：量化分析

**通用的分析公式**：
```
平均分 = Σ(所有回答分數) / 回答人數

達成率 = (達成目標的人數) / 總人數 × 100%

問題出現率 = (遇到問題的人數) / 總人數 × 100%

優先級評分 = (出現次數 × 重要性評分) / 總人數
```

### 第 3 步：定性分析

**通用的提取方法**：
```
1. 列出所有開放式回答
2. 提取關鍵詞（名詞、動詞、形容詞）
3. 統計關鍵詞出現次數
4. 排名前 3-5 的關鍵詞為「核心問題」或「核心需求」
5. 出現 ≥ 2 次的內容列為「高優先級」
```

### 第 4 步：可視化呈現

**推薦的圖表類型**：
```
評分分布 → 直方圖
各項對比 → 柱狀圖
問題排名 → 橫條圖
趨勢變化 → 折線圖
比例分布 → 餅圖
```

### 第 5 步：改進計劃

**通用的改進流程**：
```
1. 識別高優先級問題（≥ 2 人提及或評分 < 3.0）
2. 分析問題根源
3. 制定改進方案
4. 分配責任人和截止日期
5. 執行改進
6. 進行二次調查驗證
7. 記錄改進效果
```
