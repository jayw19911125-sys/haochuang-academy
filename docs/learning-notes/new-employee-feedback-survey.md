## 📊 數據分析指南

### 第 1 步：量化數據分析

**平均分計算**：
```
整體幫助程度平均分 = (Q4 所有回答的分數) / 回答人數
目標：≥ 4.0 分（「很有幫助」以上）

清晰度平均分 = (Q6 + Q15 + Q23) / 3
目標：≥ 4.0 分

推薦度（NPS）= (推薦者 - 不推薦者) / 總人數 × 100%
目標：≥ 50%
```

**分章節評分**：
```
第 11 章平均分 = (Q9 + Q11 相關) / 2
第 12 章平均分 = (Q13 + Q15) / 2
第 13 章平均分 = (Q17 + Q18 + Q19) / 3
```
