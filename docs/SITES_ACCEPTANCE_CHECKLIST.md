# 好創學院｜ChatGPT Sites 上線驗收清單

## A. Site 與存取

- [ ] Site 名稱為「好創學院」
- [ ] 已建立私有預覽或儲存版本，尚未公開
- [ ] 預設僅 Owner／指定好創成員可存取
- [ ] 非授權帳號無法進入
- [ ] 登入身份來自 workspace identity 或 ChatGPT identity
- [ ] 伺服器端使用 email 驗證授權，不使用 deviceId
- [ ] 管理員權限由伺服器端驗證

## B. 首頁與導覽

- [ ] 首頁清楚呈現「不是看完教材，是要完整跑通案件」
- [ ] 左側或主要導覽可以切換所有區域
- [ ] 不存在失效 anchor
- [ ] 桌機、平板、手機皆可使用
- [ ] 使用者可看到職務、階段、待完成模組、缺少證據、主管回饋
- [ ] 沒有 XP、排行榜、紙屑畢業動畫

## C. 第 1 個月企劃路徑

- [ ] 13 階段全部存在
- [ ] 每階段顯示目的、工具、產出與完成定義
- [ ] 包含社群全案與單線策略
- [ ] 包含市場調查與競品分析
- [ ] 包含客戶商業及受眾診斷
- [ ] 包含 IG、FB、Threads、TikTok、YouTube Shorts、LINE OA 月度矩陣
- [ ] 包含圖文、輪播、文案、短影音
- [ ] 包含發布、互動與數據優化
- [ ] 包含 AI 專案
- [ ] 包含 SKILL 建立、QA、錯誤處理與版本
- [ ] 自主學習與文書行政能力在路徑中持續可見

## D. 1／3／6 個月考核

- [ ] 正式名稱為全流程獨立執行／策略整合與專業進階／多案主責與穩定交付
- [ ] 員工畫面沒有「比其他公司強」等比較名稱
- [ ] 每階段有工作證據、門檻、獨立邊界與不可發生事項
- [ ] 第 6 個月保留連續兩個月穩定主責 3 件以上案件
- [ ] 狀態支援未開始、進行中、待審核、通過、需改善
- [ ] 需改善可記錄原因、要求與期限

## E. 工作證據與主管評核

- [ ] 使用者可以提交正本連結與說明
- [ ] Site 不複製正式檔案內容
- [ ] 主管可以批准、退回與填寫意見
- [ ] 每次評核有評核者與時間
- [ ] 可查看歷史評核紀錄
- [ ] 一般成員不能查看其他人的完整資料
- [ ] 主管可以查看團隊狀態，但必須具備管理角色

## F. D1 與資料一致性

- [ ] D1 綁定名稱為 DB
- [ ] Migration 成功執行
- [ ] 使用者身份綁定 email
- [ ] 正式進度不存 LocalStorage
- [ ] Theme／摺疊等暫時 UI 狀態可留 LocalStorage
- [ ] 重整與換裝置後正式進度仍存在
- [ ] 前端不能指定任意 email 修改他人資料

## G. 好創資料治理

- [ ] Monday 仍是 CRM／任務／專案狀態正本
- [ ] Notion 仍是 SOP／教學／知識正本
- [ ] Drive／NAS 仍是正式文件、素材與成品正本
- [ ] Slack 只作討論、通知、卡點
- [ ] LINE OA 仍是客戶正式窗口
- [ ] GitHub 仍是程式、AI 規則與版本正本
- [ ] Site 只保存入口、進度、證據連結與評核
- [ ] Site 未保存客戶帳密、API Key、正式合約、發票、銀行或大型素材

## H. 品質與安全

- [ ] 所有表單有 loading、success、error 狀態
- [ ] 空資料狀態清楚
- [ ] 可使用鍵盤操作
- [ ] Focus 可見
- [ ] 顏色對比足夠
- [ ] 支援 reduced motion
- [ ] 不含秘密值與未脫敏客戶資料
- [ ] 預覽已從一般成員與管理者兩種身份測試
- [ ] 尚未通過本清單前不得公開或擴大分享
